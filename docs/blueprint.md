# **App Name**: AquaLedger

## Core Features:

- Catch Logging: Allows fisherfolk to log details such as species, location, date, and method of catch.
- Sustainability Scoring: Shows a sustainability score for each catch, calculated using the Gemini API based on provided data.
- Sustainability Tool: Provides LLM powered recommendations for fishing location, catch method, and best practices as a tool, in order to improve the sustainability score for future catches.
- Catch List: Displays all verified, traceable catches available for buyers to view.
- Traceability View: Provides an in-depth view of each catch's journey from the sea to the buyer, including all logged details.
- Data Integrity: Ensures the accuracy and reliability of logged catch data.

## Style Guidelines:

- Refreshing light blue (#64B5F6), evoking the sea and clarity.
- Light desaturated blue (#E3F2FD), creating a calm, airy feel.
- Natural green (#81C784), reflecting sustainability and growth, placed on key action elements.
- 'Inter', a grotesque-style sans-serif providing a modern and neutral aesthetic suitable for clear communication.
- Use simple, line-style icons related to fishing, sustainability, and traceability, in the accent green color.
- A clean and intuitive layout, making data accessible and easy to understand.
- Subtle transitions and loading animations to enhance user experience without distraction.
